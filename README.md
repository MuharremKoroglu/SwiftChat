## <h2> About This App <img src = "https://c.tenor.com/JsoERRQcZqYAAAAi/thumbs-up-joypixels.gif" width = "42"></h2>
* This application is developed using Swift and RxSwift to demonstrate reactive programming principles.
* Firebase is utilized as the backend service, incorporating Firestore, Storage, and Authentication technologies.
* The app is primarily built with UIKit, but it also includes a settings page implemented with SwiftUI, showcasing the integration of SwiftUI within a UIKit project.
* Generic functions are employed to ensure versatile usage across various data structures.
* All constraints are defined programmatically using Auto Layout, providing precise control over the layout.
* LottieFiles animations are integrated throughout the application to enhance the user experience.
* Besides Firebase users, the application fetches user data from the RandomUser.me API using URLSession. This data is merged with Firebase user information to create a comprehensive contact list.
